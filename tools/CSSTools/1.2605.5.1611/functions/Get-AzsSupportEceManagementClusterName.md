# Get-AzsSupportEceManagementClusterName

## SYNOPSIS
Retrieves the name of the management cluster in the ECE infrastructure.

## SYNTAX

```
Get-AzsSupportEceManagementClusterName [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
This function queries the ECE infrastructure configuration and returns the name of the management cluster node.

## EXAMPLES

### EXAMPLE 1
```
Get-AzsSupportEceManagementClusterName
```
