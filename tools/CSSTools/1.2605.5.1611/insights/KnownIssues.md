# KnownIssues

This script checks for common known issues on Azure Local environment. The analyzer executed depends on the installed version of Azure Local.

## AzureLocal.2505

This analyzer checks for known issues related to 2505 builds.

| Rule | Synopsis |
| --- | --- |
| AzureLocal.KI.35809936 | Checks for a known issue where Authenticated Users do not have permissions to Cluster AD Computer Object |
| Windows.Network.Driver.Mellanox.251266470 | Checks to determine if Mellanox driver 25.1.26647.0 is installed |
