# Get-AzsSupportLcmDeploymentUserName

## SYNOPSIS
Retrieves the username of the Local Configuration Manager (LCM) deployment account for the Azure Local cluster.

## SYNTAX

```
Get-AzsSupportLcmDeploymentUserName [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
This function retrieves the username of the Local Configuration Manager (LCM) deployment account for the Azure Local cluster by parsing the ECE cloud definition XML.

## EXAMPLES

### EXAMPLE 1
```
Get-AzsSupportLcmDeploymentUserName
```
