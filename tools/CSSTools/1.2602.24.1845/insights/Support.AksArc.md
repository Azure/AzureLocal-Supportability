## Support.AksArc

This script checks the state of the AKS ARC.

| Rule | Synopsis |
| --- | --- |
| Windows.System.Service.State | This script checks the state of a service on the system. |
| Support.AksArc.CloudAgentAffinityRule | Validate Cloud Agent Cluster Affinity Rule |
| Support.AksArc.MocCertificateExpired | Validate certificates for MOC not expired |
| Support.AksArc.MocCloudAgentNotRunning | Check if the MOC Cloud Agent service is running in a failover cluster. |
| Support.AksArc.MocConfig | Check if the MOC admin token is expired. |
| Support.AksArc.MocIdentityTokensEmpty | Checks for empty MOC identity tokens. |
| Support.AksArc.MocUpdating | Validate MOC not stuck in updating state |
| Support.AksArc.MocVersion | Check if the MOC Cloud Agent version is on the latest patch. |
| Support.AksArc.NodeAgent.PendingDeletion | Check if the MOC Cloud Agent Service is marked for deletion. |
| Support.AksArc.MissingMocPsRegistry | Tests if the MOC PowerShell registry configuration is missing. |
