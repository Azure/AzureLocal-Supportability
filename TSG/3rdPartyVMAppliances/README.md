# TSG and Reference Documents for Third-Party VM Appliances on Azure Local

This section covers troubleshooting guidance and interoperability notes for third-party virtual appliances hosted as Hyper-V guests on Azure Local — for example, virtual load balancers, virtual firewalls, and similar network or security appliances. These guides describe interop considerations, supported configurations, and vendor-issued fixes that affect how those appliances behave on the Azure Local Hyper-V platform.
